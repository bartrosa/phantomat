# Phantomat

[![CI](https://github.com/OWNER/phantomat/actions/workflows/ci.yml/badge.svg)](https://github.com/OWNER/phantomat/actions/workflows/ci.yml)
[![License](https://img.shields.io/badge/license-Apache%202.0-blue.svg)](LICENSE)

GPU-oriented visualization engine for linked exploration of data in 2D/3D, geospatial, and orbital reference frames—**Rust** core, **Python-first** API, thin web bindings.

Replace `OWNER` in the badge URL with your GitHub user or organization after you publish the repository.

## Quick start (Rust)

Requires **Rust 1.82** (see `rust-toolchain.toml`) and [rustup](https://rustup.rs/).

```bash
git clone https://github.com/OWNER/phantomat.git
cd phantomat
cargo build --workspace
cargo test --workspace
```
